# Space invaders

remake for [vim](https://www.vim.org/) users

### Installation

1. clone repository

2. install requirements

```bash
pip install -r requirements.txt
```

### Using

| key | action  |
| :-: | :-----: |
| h   | left    |
| j   | fire    |
| k   | nothing |
| l   | right   |

Also u can start game with custom value of rows

```bash
python3 main.py 10
```
